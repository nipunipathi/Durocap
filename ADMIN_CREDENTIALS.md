## Troubleshooting

### Can't Find Admin Login?
- Scroll to the **bottom** of any page
- Look for **"Quick Links"** section in footer
- Click **"Admin Login"** link

### Login Not Working?
- Make sure you're using **username**, not email
- Username: `admin` (lowercase)
- Password: `admin123` (lowercase, no spaces)
- Check for typos

### Logged Out Automatically?
- Session may have expired
- Simply login again with the same credentials
