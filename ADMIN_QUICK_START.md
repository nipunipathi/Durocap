## ğŸš¨ Troubleshooting

### Problem: Project not showing in carousel
**Solution:** 
- Check if project is marked as "Featured"
- Click star icon to feature it
- Refresh homepage

### Problem: Image not displaying
**Solution:**
- Verify image URL is correct
- Test URL in browser
- Ensure image is publicly accessible
- Check image preview in edit dialog

### Problem: Can't delete project
**Solution:**
- Ensure you're logged in as admin
- Check browser console for errors
- Try refreshing the page

### Problem: Changes not appearing
**Solution:**
- Refresh the page (Ctrl+R or Cmd+R)
- Clear browser cache
- Check if changes were saved (look for success toast)
